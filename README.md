# CV Design Variants Source

Source files and generated PDFs for multiple CV design variants.

## Included
- PDF variants A-G
- HTML sources for polished variants D-G
- Generation scripts:
  - `generate-cv.js`
  - `generate-cv-variants.js`
  - `generate-cv-html-style-variants.js`
  - `render-html-to-pdf.js`

## Render HTML to PDF
```bash
npm install
node render-html-to-pdf.js Reinis_Allis_CV_Variant_D_HTMLPolished.html Reinis_Allis_CV_Variant_D_HTMLPolished.pdf
```
